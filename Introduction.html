About ...


WebBook is a collection of learning materials. It is a web application to create amaging teaching items, small applets for interesing exercises. No more boring texts. Otherwise, now you are getting intresting objects, with which now you can iteract. Not only this, now you can enjoy learning with inline-fun instead of doing boring exercises.
WebBook is a new evolution for book lovers. Printed book > PDF > WebBook.
Take a look at the screenshot(quite...).



How To ???


WebBook version 1.0.1 is a small web app. It needs contribution from others to develop it to a complete WebBook experience. In order to do that i have publised this book on how to get your hands on web application development for WebBook(1.0.1).
So, learning web application programming using WebBook! Intersing, isn't it ?

Developper

As a developper you can desing and style the app, or make it interactive as a scriptter. As a developper you should know HTML5 or CSS or Javascript. If you don't know any of this but you are an artist then we need you too. To make the app attractive we need artists (really in need of one !). Moreover, if you have any good idea you are welcome too.

Note: For detail about the app construction see the help section.

Tester

As a tester you will test our app for bugs, so that we(developper) can fix them and make the app bug free.

Book Writer

Book writers can use this app to publish their WebBook. This platform is mainly for science and tecnology related books. But other kind of writings are also welcome. You can really become creative with it. Book writers will have to know about web application programming but not as much as developpers. If you know webpage desing you can write a WebBook.

Note: As a writter you should not alter the core application interface.

Reader

I wish readers will have a great experience with WebBook. We will look forward for any suggestions and advices from you.
